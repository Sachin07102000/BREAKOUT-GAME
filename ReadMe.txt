## Introduction

This is a simple Breakout game created using HTML, CSS, and JavaScript. The goal of the game is to break all the bricks using the ball and the paddle while avoiding letting the ball hit the bottom of the game board.

## How to Play

To play the game, follow these steps:

1. Use the left and right arrow keys to control the paddle.
2. Bounce the ball off the paddle to break the bricks.
3. Try to clear all the bricks to advance to the next level.
4. If the ball hits the bottom of the board, the game is over.
5. You can restart the game by pressing the 'Space' key.

## Controls

- Left Arrow Key: Move the paddle to the left.
- Right Arrow Key: Move the paddle to the right.
- Space Key: Restart the game when it's over.

## Game Mechanics

- The paddle is controlled by the player and is used to bounce the ball.
- The ball moves in the direction indicated by its velocity.
- Bricks must be broken by bouncing the ball off them.
- When all bricks are broken, you advance to the next level.
- The game is over when the ball hits the bottom of the board.

## Features

- Basic Breakout game with bricks, paddle, and ball.
- Multiple levels with increasing difficulty.
- Score tracking to keep track of your progress.
